# PraxisKMM

Minimal **Kotlin Multiplatform** project with SwiftUI, Jetpack Compose, Compose for Wear OS,. Currently running on
* Android (Jetpack Compose)
* Android App Widget
* Wear OS (Compose for Wear OS)
* iOS (SwiftUI)
* iOS App Widget (SwiftUI)
* watchOS (SwiftUI)
* macOS (SwiftUI)
